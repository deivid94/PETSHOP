🐾 PETSHOP — Agendamento Web de Serviços
💡 A história por trás do projeto
Todo dono de pet já passou por isso: ligar para o petshop, esperar alguém atender, tentar encaixar um horário no meio da correria do dia — tudo isso para agendar um simples banho. Demorado, frustrante e desnecessário.
Foi pensando nessa dor real que o PETSHOP nasceu: uma aplicação web moderna que permite ao tutor agendar serviços para seu pet de forma rápida, simples e online, sem precisar sair da tela.


- **Programming Language:** JavaScript
- **Package Manager:** Npm

### Installation

Build PETSHOP from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/deivid94/PETSHOP
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd PETSHOP
    ```

3. **Install the dependencies:**

[![npm][npm-shield]][npm-link]

**Using [npm](https://www.npmjs.com/):**
```sh
❯ npm install
```

[npm-shield]: https://img.shields.io/badge/npm-CB3837.svg?style=for-the-badge&logo=npm&logoColor=white
[npm-link]: https://www.npmjs.com/

### Usage

Run the project with:

**Using [npm](https://www.npmjs.com/):**
```sh
npm start
```

### Testing

Petshop uses the {__test_framework__} test framework. Run the test suite with:

**Using [npm](https://www.npmjs.com/):**
```sh
npm test
```
